# 🚀 #Strategic Agency Landing Page

A responsive strategic agency landing page built according to a modern design concept.  
The project focuses on clean layout structure, scalable styling, and modern frontend practices inspired by real-world agency websites.

---

## 🚀 #Features
- Fully responsive layout (desktop, tablet, mobile)
- Pixel-perfect implementation based on design
- Clean and modern UI for agency presentation
- Structured sections (services, strategy, about, contact)
- SCSS for scalable and maintainable styling
- BEM methodology for consistent CSS architecture
- Smooth and user-friendly layout flow

---

## 🛠️ #TechStack
- HTML5
- SCSS (Sass)
- JavaScript
- BEM methodology
- npm

---

## 🎯 #Purpose
This project was created to practice building modern agency landing pages, improve layout structuring skills, and apply scalable CSS architecture in real-world UI scenarios.

cd Stategic-agency-landing-page

### 📦 Install dependencies
npm install
# or
yarn install

### ▶️ Run the project locally
npm start
# or
